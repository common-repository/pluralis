Improve your landing page conversions by using the power of the crowd with Pluralis

== Description ==
This plugin lets you use Pluralis' platform on wordpress as well.

If you don't know who are we and what is Pluralis, visit our site @ www.pluralis.com


== Installation ==
1. Upload "pluralis-plugin.php" to the "wp-content/plugins" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Use the admin panel in the settings menu to set the tag number that was given to you on the pluralis web app (at the very end of the contest creation wizard).
4. Validate your tag in Pluralis' contest creation wizard

== Screenshots ==
1. Go to Pluralis Plugin Admin Options.
2. Enter the number you got at the end of the Pluralis contest creation wizard in the appropriate field and hit submit.

== Frequently Asked Questions ==

= Where do I find the tag number? =

Go into Pluralis web app and enter your contest. At the launch page you'll see the tag.
If the tag is <b>&lt;script type="text/javascript" src="//cache.pluralis.com/srv/juggler_1032.js"&gt;&lt;/script&gt;</b>, then your tag number is 1032.
